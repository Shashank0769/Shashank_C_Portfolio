Stock Prediction using TensorFlow

Project Overview

The Stock Prediction using TensorFlow project is designed to predict the next day's stock price using a machine learning model. The system fetches real-time stock data, processes it, and utilizes a trained model to provide accurate stock price forecasts.

Key Features

  • Fetches real-time stock data from Yahoo Finance (`yfinance`).
  • Utilizes TensorFlow to train a predictive model.
  • Provides an interactive Streamlit UI for user input.
  • Predicts the next day's stock price dynamically.

Workflow & Methodology

    Data Collection & Preprocessing

    • Historical stock data is retrieved using the `yfinance` API.
    • Data is cleaned, missing values are handled, and features are engineered for training.

    Model Training & Prediction

    • A simple TensorFlow model is trained to predict stock prices.
    • Performance is validated using error metrics such as RMSE.

    Visualization & Insights

    • Predicted vs actual stock prices are displayed using interactive charts.
    • Users can analyze stock trends based on past data.

    Deployment & Real-Time Predictions

    • The application is deployed using Streamlit Cloud, allowing easy access.
    • Users can enter a stock symbol, choose a date range, and predict the next day's price.

Technologies Used

  • Python
  • TensorFlow
  • yfinance
  • Pandas, NumPy
  • Streamlit (for deployment)
