Crowd Detection Using OpenCV

Project Overview

The Crowd Detection Using OpenCV project implements a detection system using YOLOv3 to identify crowds in video frames. The system logs crowd events and saves results for further analysis.

Key Features

  • Detects persons in video frames using YOLOv3.
  • Identifies crowd events based on predefined criteria.
  • Logs detected crowd events, including frame numbers and person count.
  • Saves results in a CSV file for analysis.

Workflow & Methodology

    Data Collection & Processing

    • Processes video frames using OpenCV.
    • Detects persons using YOLOv3.

    Crowd Detection Criteria

    • A crowd event is detected if three or more persons stand close for 10 consecutive frames.
    • Logs frame numbers and crowd size in a CSV file.

    Deployment & Usage

    • Requires YOLOv3 weights & config files.
    • Run `detection.py` to analyze a video file.

Technologies Used

  • Python
  • OpenCV
  • YOLOv3
  • NumPy, Pandas
